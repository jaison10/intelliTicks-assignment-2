### Hello <img src="https://media.giphy.com/media/hvRJCLFzcasrR4ia7z/giphy.gif" width="25px">

Install the following:
<ul>
  <li>body-parser</li>
  <li>express</li>
  <li>airtable</li>
  <li>nodemon</li>
</ul>

</br>

Replace the API key and table id in ```app.js``` file.  
</br>
</br>

Inside package.json, set the script as:
```bash
"dev": "nodemon ./app.js",
```
</br>

## Usage:
```bash
npm run dev
```
